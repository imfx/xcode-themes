## Installation
1. Copy the theme(s) to the Xcode's `FontAndColorThemes` folder (`~/Library/Developer/Xcode/UserData/FontAndColorThemes/`).
2. Restart Xcode.
3. Go to `Xcode > Preferences > Fonts & Colors`.
4. Select your favorite theme.
